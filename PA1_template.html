Devise a strategy for filling in all of the missing values in the dataset

Use a for loop to run through each row and where the value is NA replace with the mean for the number of steps for that time interval

fullact <- act
for (i in 1:nrow(fullact)) {
  fa <- fullact[i,]
    if(is.na(fa$steps)) {
      a <- fullact[fullact$interval==fa$interval,]
      b <- mean(a$steps, na.rm=TRUE)
      fullact[i,1] = b
    }
}
